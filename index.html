About

Selamat datang di website pribadi saya. Website ini dibuat sebagai media untuk memperkenalkan diri sekaligus mendokumentasikan perjalanan saya selama menjalani kehidupan sebagai mahasiswa. Di sini saya membagikan sedikit cerita tentang diri saya, aktivitas sehari-hari, galeri foto, hingga beberapa momen yang menurut saya berkesan.

Semoga melalui website sederhana ini, teman-teman dapat mengenal saya lebih dekat. Terima kasih sudah meluangkan waktu untuk berkunjung.

Profile

Hallo,saya Endang Salsabila

Saya lahir pada 12 Juli 2006 di Kecamatan Sungkai Utara, Kabupaten Lampung Utara. Saya merupakan anak pertama dari empat bersaudara.Saat ini saya sedang menempuh pendidikan di Universitas Muhammadiyah Kotabumi, Program Studi Sistem dan Teknologi Informasi. Saya memiliki ketertarikan pada dunia teknologi, khususnya dalam bidang pemrograman dan pengembangan website. Di luar kegiatan perkuliahan, saya senang menonton film, mendengarkan musik, serta menghabiskan waktu bersama teman-teman. Saya percaya bahwa setiap pengalaman adalah bagian dari proses belajar untuk menjadi pribadi yang lebih baik.
